### 通用
- 查看某个类型的字段结构
```bash
ros2 interface show geometry_msgs/msg/Pose
```

### 话题 topic
##### 最小实现
- 订阅者
QOS可以填10
```py
self.sub = self.create_subscription(
    TYPE,
    NAME,
    CALLBACK,
    QOS,
)
```

- 创建发布者
QOS可以填10
```py
self.pub = self.create_publisher(TYPE, TOPIC_NAME, QOS)
```

- 发布话题
```py
msg = TYPE()
msg.FIELD1​ = DATA
msg.FIELD2​ = DATA

self.pub.publish(msg)
```

##### 命令
- 查看
```bash
ros2 topic list
```

- 查看信息
```bash
ros2 topic info 话题名
```

- 查看详细信息
```bash
ros2 topic info 话题名 -v
```

- 打印
```bash
ros2 topic echo 话题名
```

- 打印一次
```bash
ros2 topic echo 话题名 --once
```

- 测量频率
```bash
ros2 topic hz 话题名
```

- 使用命令行发布某个话题
```bash
ros2 topic pub topic_name message_type values
```
