### 话题 topic
##### 最小实现
- 订阅者
QOS可以填10
```py
self.sub = self.create_subscription(
    TYPE,
    NAME,
    CALLBACK,
    QOS,
)
```

- 创建发布者
QOS可以填10
```py
self.pub = self.create_publisher(TYPE, TOPIC_NAME, QOS)
```

- 发布话题
```py
msg = TYPE()
msg.FIELD1​ = DATA
msg.FIELD2​ = DATA

self.pub.publish(msg)
```

- 发布静态tf
```py
self.static_tf_broadcaster = StaticTransformBroadcaster(self)
```

- 发布动态tf
```py
self.dynamic_tf_broadcaster = TransformBroadcaster(self)
NAME = TransformStamped()
NAME.header.stamp = self.get_clock().now().to_msg() 
NAME.header.frame_id = NAME
NAME.child_frame_id = NAME
self.dynamic_tf_broadcaster.sendTransform(NAME)
```

- 响应服务(响应端)
```py
self.srv = self.create_service(TYPE, NAME, CALLBACK)
```

- 发布服务，处理响应(发布端)
```py
self.cli = self.create_client(TYPE, NAME)
while not self.cli.wait_for_service(timeout_sec=Time):
    return
self.send_request()
def send_request(self):
    req = TYPE.Request()
            future = self.cli.call_async(req)
            future.add_done_callback(CALLBACK)
```

##### 命令
- 查看
```bash
ros2 topic list
```

- 查看信息
```bash
ros2 topic info 话题名
```

- 查看详细信息
```bash
ros2 topic info 话题名 -v
```

- 打印
```bash
ros2 topic echo 话题名
```

- 打印一次
```bash
ros2 topic echo 话题名 --once
```

- 测量频率
```bash
ros2 topic hz 话题名
```

- 使用命令行发布某个话题
```bash
ros2 topic pub topic_name message_type values
```

### 服务 service

##### 命令
- 查看有哪些服务
```bash
ros2 service list
```

- 查看某个服务的类型
```bash
ros2 service type 服务名
```
- 调用某个服务的类型
```bash
ros2 service call 服务名 std_srvs/srv/服务类型 "{参数名：参数}"
```

##### 动作 action

##### 命令
查看有哪些动作
```bash
ros2 action list
```

```bash
ros2 action info
```
