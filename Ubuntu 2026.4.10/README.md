#### linux常用终端命令
##### 查看终端所在位置下有什么文件和文件夹
```bash
ls
```

##### 切换终端所在位置
以下的所有命令结尾都可以加```/```,结尾的```/```对命令的结果没有影响
如```cd /opt/ros```和```cd /opt/ros/ ```

- 切换到某个绝对路径 绝对路径以```/```开头
    ```bash
    cd /opt/ros/
    ```

- 切换到终端位置下的某个位置 ```.```表示当前文件路径,可省略
    ```bash
    cd ./a/b/c
    ```

- 向上一级切换 使用```..```
    ```bash
    cd ./a/b/c/..
    ```
    这个的命令相当于切换到b文件夹
    ```bash
    cd ..
    ```
    这个终端所在路径的上一级

##### 管理员命令
